核心功能

围绕抖音公开内容整理流程设计：先按关键词找作品，再整理评论、达人主页、视频文件和链接格式。

关键词

搜索作品采集

按关键词搜索抖音作品，整理标题、作者、发布时间和互动数据。

评论

评论采集

采集指定作品或搜索结果作品下的评论内容、评论者和互动信息。

主页

主页作品采集

输入达人主页链接，批量整理公开作品列表，并可按需下载视频文件。

转换

链接与 UID 转换

支持主页链接、抖音号、UID、App 分享链接和 PC 链接之间的常见格式转换。

适合哪些使用场景

页面只描述公开数据整理和导出能力，不承诺绕过平台权限、风控或访问限制。

关键词选题调研了解某个关键词下的相关作品、标题方向、互动数据和发布时间。
评论线索整理批量导出评论内容、评论者信息和评论时间，便于后续处理。
竞品内容观察跟踪同行账号的公开作品列表、更新节奏和互动表现。
达人主页盘点整理指定达人主页的公开作品、发布时间、互动表现和视频文件。
链接格式处理将 App 分享链接、PC 链接、主页链接、抖音号和 UID 做统一整理。
公开作品归档保存公开作品的基础信息、视频直链或本地视频文件，便于内部复盘。

常见问题

这里回答购买前最常见的问题。更细的字段说明、视频下载说明和反馈格式请查看 GitHub README。

是否需要安装 Python？

不需要。软件已经打包为桌面客户端，下载对应系统版本后即可运行。

Cookie 是做什么用的？

Cookie 用于让软件以当前账号状态访问平台公开数据。请使用自己的抖音账号 Cookie。

视频下载失败怎么办？

请检查作品链接是否有效、网络是否正常、输出目录是否可写，以及当前账号是否具备访问该作品的权限。

采集中断后数据会丢失吗？

软件会在采集过程中持续保存结果，已完成部分通常会保留在 CSV 文件中。

准备整理抖音公开数据？

先看演示或下载最新版，也可以直接进入商品页开通授权。
