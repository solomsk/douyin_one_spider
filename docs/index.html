👋 软件简介 👋 Overview

douyin_one_spider 是一款面向抖音数据采集场景的桌面 GUI 工具。 支持关键词搜索作品采集、指定作品评论采集、主页作品批量采集、链接/UID 互转等功能， 采集结果自动保存为 CSV 文件并支持下载视频。用户无需配置 Python 环境，下载客户端后登录即可使用。

douyin_one_spider is a desktop GUI tool for Douyin (TikTok China) data collection. Supports keyword-based post search, comment scraping, profile post crawling, link/UID conversion, and more — with CSV export and video download. No Python setup required.

❓ 常见问题 ❓ FAQ

🟢 是否需要安装 Python？
不需要。软件已打包为桌面客户端，下载对应系统版本后即可运行。
🟢 Do I need to install Python?
No. The software is packaged as a standalone desktop app — just download and run.
🟢 如何配置 Cookie？
软件内置 Cookie 小工具，支持扫码登录获取或手动粘贴 Cookie 字符串，操作简单。
🟢 How do I configure the cookie?
The app has a built-in cookie utility — scan QR to log in or manually paste your cookie string.
🟢 采集中断后数据会丢失吗？
不会。软件在采集过程中每页即时保存结果，网络异常或任务中断不会丢失已采集的数据。
🟢 Will I lose data if crawling is interrupted?
No. Results are saved per page instantly — network issues or interruptions won't wipe collected data.
🟢 支持导出哪些格式？
支持导出 CSV 文件和视频文件。CSV 可用 Excel / WPS / Numbers 直接打开，视频为无水印 MP4 格式。
🟢 What export formats are supported?
CSV files and video files. CSV opens in Excel / WPS / Numbers. Videos are watermark-free MP4 format.
